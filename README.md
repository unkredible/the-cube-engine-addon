# The Cube Engine Addon

Public Blender addon package for The Cube Engine.

## Install

1. Download `ai_render_finisher.zip` from the latest GitHub release.
2. Open Blender.
3. Go to `Edit > Preferences > Add-ons > Install...`.
4. Select the ZIP.
5. Open the `The Cube` sidebar tab in the 3D View.
6. Generate an addon key from the dashboard and paste it in addon Preferences.

## Backend

The addon does not contain provider keys, production prompts, billing logic or protected AI routing. AI jobs are authorized and processed by The Cube Engine backend after addon key and subscription checks.
